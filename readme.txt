== Description ==

This plugin is used to show the google picasa web albums. This plugin also provides the shortcode [wp-google-picasa-web].
This plugin gets google picasa web account key and then generates the list of all available albums. You can also watch the images from specific album.

YOu can use this plugin to list out all the images from picasa acount.


== Installation ==

Way 1:
1. Upload `wp-google-picasa-web.zip` to the `/wp-content/plugins/` directory
2. Extract the zip file.
3. Activate the plugin from wordpress plugins page. And you have done.

Way 2: 
1. Go to plugins page in admin.
2. Click on Add New button.
3. Search for "WP Google Picasa Web".
4. Click install on WP Google Picasa Web.
5. Activate the plugin.
